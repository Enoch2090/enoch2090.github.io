草稿处理的无缝衔接：Drafts 终于登上 macOS

Enoch2090
2019-03-27
Drafts

Drafts 对双平台的支持意味着无限可能。在此前，我们一直认为 Drafts 更多只是一个文字的收集端，你可以随时打开它记点东西，然后一键把记下的东西发送到别的地方。但那以后我们几乎就不会再去管已经存档的文本。DFM 的出现，不仅仅是简单地补全了 Drafts 这个应用在桌面端的表现。
