在此集合所有我个人折腾出来的小项目，以时间为先后排列。

  • 20200716 StellarConvergence
    一个由 Python 编写，基于 Streamlit 的简单应用，用于在地图上寻找数个地点的中点。
  • 20200615 pdfDropDuplicates
    通过图像哈希算法剔除 pdf 中的重复页面。
  • 20190908 Keyboard Resources
    客制化键盘的一些资源。介绍文章：客制化机械键盘入门指南（有付费墙）。
  • 20190628 Drafts Document
    Drafts 文档的中文版本。目前我仍在翻译中，内容会逐步更新。 此文档也在 Github 同步更新。
  • 20190503 Drafts 统计器
    一个用于统计 Drafts 草稿数据的小工具。
  • 20190226 Phillip’s Gadgets
    一些自己编写的效率小工具，内容包括了我常用的效率应用：Keyboard Maestro，Drafts……
  • 20180815 Wake up, Opportunity!
    虽然火星超级尘暴已经过去了，但是火星上的机遇号探测器 (Opportunity) 由于还没有充够电，并没有醒来和地球上关心它的人取得联系。美国喷气推进实验室的科学家们为了给自己和 Oppy 打气，专门做了一个叫“机遇号，醒来！” (Wake Up, Opportunity!) 的 Spotify 播放列表。我在网易云音乐上做了一个搬运版本。
    不过最终，机遇号还是没有醒来。这就是生活。
